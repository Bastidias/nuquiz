# Security Incident Types

**Topic:** 2 — Safety and Infection Prevention and Control &nbsp;|&nbsp; **Pattern:** Dimensions &nbsp;|&nbsp; **Tags:** 2.8, 2.2, IP-CD, IP-CJ, state-variance
**Status:** draft (SME review pending)

The six security incident types most often tested under NCLEX 2.8 (Security Plan) and 2.2 (Emergency Response Plan): infant abduction, active shooter, workplace violence, elopement, bomb threat, and hostage situation. Sibling Concept `hospital-emergency-codes.md` covers the same events from the code-color taxonomy angle (trigger meaning, who responds, escalation); this Concept focuses on **incident response** — what the bedside nurse does, when the facility goes into lockdown, and how the event is reported afterward. Tagged `state-variance` because the operative definitions, lockdown thresholds, and reporting destinations are determined by state hospital association policy, state Department of Health, and individual facility security plans rather than by a federal standard. NCLEX continues to test the predominant US conventions captured below.

## Notes

- **Cell convention:** each `<br>`-separated item is one atomic Fact. No parentheticals in cells.
- **No federal lockdown standard.** No federal regulation defines hospital lockdown thresholds. The Joint Commission Emergency Management standards require an emergency operations plan that addresses security incidents, but specific lockdown triggers are facility-defined [s6, s7]. NCLEX items reflect the predominant US convention (lockdown for Code Pink, Code Silver, hostage; not routinely for Code Gray or Code Green) — captured above and tagged `state-variance` because the operative rules vary by hospital association, state Department of Health, and individual facility policy.
- **Sibling Concept link.** Sibling `hospital-emergency-codes.md` (Topic 2.2) covers the same six incident triggers from the code-color taxonomy angle — its columns are trigger, immediate nurse action, who responds, escalation. This Concept uses different columns (lockdown criteria, post-event reporting) and focuses on incident response under NCLEX 2.8 (Security Plan). Rows overlap intentionally; both Concepts are warranted because the column sets ask different exam questions.
- **Run / Hide / Fight.** The Department of Homeland Security Run / Hide / Fight protocol is the dominant US framework for active shooter response and is referenced in CDC NIOSH workplace-violence guidance for healthcare settings via the HHS guide "Incorporating Active Shooter Incident Planning into Health Care Facility Emergency Operations Plans" [s4]. Hide and Barricade are listed as separate Facts because students often conflate them — Hide means concealing oneself; Barricade means physically obstructing the door.
- **NIOSH de-escalation framing.** The Code Gray (workplace violence) immediate-nurse-action cell mirrors the NIOSH 2002-101 sequence: present calm caring attitude, acknowledge feelings, avoid matching threats, keep exit path open, remove self, call security [s3]. The order is deliberate — de-escalation precedes withdrawal precedes calling security.
- **Joint Commission sentinel event triggers.** The Joint Commission lists infant abduction, infant sent home with wrong parent, rape, assault leading to death or permanent loss of function, homicide on hospital property, and elopement resulting in death or major permanent loss of function as reviewable sentinel events [s7]. Reporting to the Joint Commission is strongly encouraged but not required. Internal incident reporting and OSHA Form 300 entries are independent of the sentinel event policy.
- **OSHA Form 300.** OSHA recordkeeping requires logging work-related injuries and illnesses on Form 300 [s5]. Workplace-violence injuries to staff are loggable events; the cells flag this for the Code Silver, Code Gray, and Code Black rows where staff injury is plausible. Mental-health-only sequelae such as PTSD are not loggable unless an employee voluntarily provides a clinician opinion linking the condition to the work event [s5].
- **AMBER alert.** AMBER alert is a Department of Justice program for missing children. Its activation in a Code Pink response is conventional pedagogy and varies by state criteria; flagged `[needs source]` because no allowlisted primary source ties AMBER directly to a hospital Code Pink trigger.
- **Code-color variability.** Hospital emergency codes are not federally standardized in the United States. The Healthcare Association of Southern California (HASC) 2014 guide is the dominant US reference [s8]; state and individual hospital adoption varies, particularly for security-related codes (Silver, Gray, Black) and for Code Green (some hospitals use Code Walker or plain-language). The trigger-code column reflects the HASC-anchored convention NCLEX items mirror.
- **Out of scope.** Tactical law-enforcement doctrine, bomb-squad protocols, AMBER alert state-level activation criteria, the full text of facility emergency operations plans, hostage negotiation specifics, and post-incident debriefing structure.
- **Tag note.** Carries `IP-CD` (Communication and Documentation) because every incident response includes overhead paging, security notification, and incident-report filing — communication-and-documentation acts. Carries `IP-CJ` (Clinical Judgment) because nurse decisions about lockdown timing, when to escalate Gray to Silver, and when to call law enforcement vs handle internally are clinical-judgment acts. Tag flexibility rule applied — both `IP-CD` and `IP-CJ` go beyond the README spec but are warranted by content. Carries `state-variance` per the no-federal-standard reasoning above.

### Tricky distractors

- **Code Gray vs Code Silver.** The most common code-color confusion. Gray = workplace violence without a weapon; Silver = active shooter or weapon involved. Wrong-answer pattern: calling Code Gray when a firearm has been identified, or activating facility lockdown on a Code Gray. Lockdown is reserved for Code Silver.
- **Run / Hide / Fight order.** Run when a safe egress route exists; Hide when egress is blocked; Fight only as last resort. Wrong-answer pattern: hiding when an exit is clearly available, or fighting before exhausting Run and Hide.
- **Elopement search vs facility lockdown.** Elopement triggers a search, not a lockdown. Wrong-answer pattern: locking down the facility when a confused patient leaves the unit. Code Pink (infant abduction) does trigger lockdown; Code Green (elopement) does not.
- **Bomb threat — what NOT to do.** Do not use radios or cell phones near a suspicious package; the radiofrequency may detonate the device. Do not pull the fire alarm. Wrong-answer pattern: activating the fire alarm to evacuate, or using the unit phone next to the suspicious package.
- **Hostage situation — do not negotiate.** Bedside nurses and first responders do not initiate negotiation; that is law enforcement's role per facility hostage incident plans. Wrong-answer pattern: attempting to talk down the hostage-taker or making demands.
- **Code Pink lockdown scope.** Code Pink locks down the entire facility, not just the unit. The infant may have been moved off-unit by the time the code is called. Wrong-answer pattern: locking down only the maternity floor.
- **Sentinel event vs incident report.** Internal incident reports are filed for every security event. Joint Commission sentinel event reports are reserved for specific outcomes (death, permanent loss of function, infant abduction, elopement with bad outcome). Wrong-answer pattern: filing a sentinel event report for every workplace violence verbal threat, or skipping the internal incident report because the event "wasn't a sentinel event."
