# Hospital Emergency Codes

**Topic:** 2 — Safety and Infection Prevention and Control &nbsp;|&nbsp; **Pattern:** Dimensions &nbsp;|&nbsp; **Tags:** 2.2, IP-CD, state-variance
**Status:** draft (SME review pending)

The seven color codes most often tested under NCLEX 2.2 (Emergency Response Plan) for overhead-paged hospital emergency events. There is no federal standard for hospital emergency code colors in the United States [s1, s2]; the most-cited reference is the Healthcare Association of Southern California (HASC) "Healthcare Facility Emergency Codes: A Guide for Code Standardization," first published in 2000 after a fatal shooting at a Southern California medical center where the wrong code was called, and revised in 2014 [s1, s3]. State and individual health-system adoption varies — peer-reviewed surveys consistently document Code Blue (cardiac arrest) and Code Red (fire) as nearly universal across US hospitals, with substantial variance for security/violence codes (Silver, Gray, Black) and hazardous-material codes (Orange) [s2, s4]. The cells below capture the HASC convention because it is the dominant US reference and the framework NCLEX items mirror; the unsourced-values table flags every cell where the value is conventional NCLEX-pedagogy framing rather than a citable primary source.

|---|---|---|---|---|

## Notes

- **Cell convention:** each `<br>`-separated item is one atomic Fact. No parentheticals in cells.
- **No federal standard.** Hospital emergency codes are not federally standardized in the United States. The Joint Commission requires a hospital to define and rehearse an emergency-codes plan but does not specify which colors mean what [s2]. The HASC 2014 guide is the dominant US reference and the de facto NCLEX framework, but state hospital associations have diverged — at least 12 state associations recommend plain-language alerts instead of color codes [s2]. Tagged `state-variance` for this reason.
- **Why HASC anchors the cells.** The HASC 2000 standard arose from a fatal incident: three persons were killed at a Southern California medical center after the wrong emergency code was overhead-paged [s3]. The committee's 2000 publication was revised in 2014 and adopted across most California hospitals; subsequent peer-reviewed surveys of hospitals in other states show partial adoption with Blue/Red/Pink most consistent and Silver/Gray/Black variable [s2, s4]. NCLEX items are written to the consistent national pattern rather than to any single state's exact list.
- **Code Blue clinical framing.** Code Blue is synonymous with cardiopulmonary arrest in nearly all Western healthcare settings [s5, s6]. The historical attribution is to Hughes Day, a Kansas cardiologist who introduced the term after observing patients with severe cardiopulmonary failure displaying cyanosis [s5]. The code-team composition cell reflects the AHRQ PSNet case discussion of standard responding personnel [s4]; specific roles vary by facility.
- **Code Pink — pediatric variants.** The HASC convention reserves Pink for infant abduction (under 1 year) and Purple for child abduction (1+ year) [s2]. Some hospitals use Pink for both — the cell uses the HASC narrower scope.
- **Code Silver — Run/Hide/Fight.** The Department of Homeland Security's Run/Hide/Fight protocol is the dominant US framework for active-shooter response and is referenced in CDC NIOSH workplace-violence guidance for healthcare settings [s7]. The protocol is not in itself a NIOSH document; NIOSH cites the HHS healthcare-facility active-shooter planning guide [s7].
- **Code Gray vs Code Silver.** Code Gray is used for a combative person without a weapon; Code Silver escalates when a weapon is involved or shots are fired [s4]. The CDC NIOSH workplace-violence guidance underlies the "remove self / call security / report" Code Gray nurse action [s7]; the action transitions to Run/Hide/Fight if the situation escalates to Code Silver.
- **Code Black variance.** Code Black means bomb threat in the HASC convention [s2, s4]. The Riyadh variability study documented seven different meanings for Code Black across hospitals studied, including external disaster, internal disaster, mass casualty, combative person, and patient death [s2]. The HASC bomb-threat meaning is the conventional NCLEX answer; the action and responder cells are the bomb-threat protocol and are flagged as unsourced because no allowlisted primary source enumerates the specific bedside-nurse first actions for a bomb threat at the granularity needed.
- **Plain-language alternative.** A growing minority of US health systems and 12 state hospital associations recommend plain-language alerts ("Code Red" → "Fire alarm, north wing, evacuate") rather than color codes, citing better integration with the National Incident Management System (NIMS) [s2]. NCLEX continues to test the color-code convention because it is the current dominant practice.
- **Sibling Concept link.** Sibling `security-incident-types.md` (Topic 2.8) covers the same security events (infant abduction, active shooter, workplace violence, bomb threat) from a security-plan / incident-trigger angle — that Concept's columns are trigger code, immediate nurse action, lockdown criteria, post-event reporting. This Concept's columns differ enough that both are warranted; cross-tag `2.2` and `2.8` if SME requests merge. Sibling `disaster-triage-start-jumpstart.md` covers triage tagging that follows a Code Triage activation. Sibling `fire-response-race-pass.md` covers RACE and PASS in detail; the Code Red row here points to RACE without enumerating its steps.
- **Out of scope.** The actual content of facility-specific emergency operations plans, the bomb-squad protocol, the SWAT tactical doctrine, and post-incident debriefing structure are out of scope for this Concept.
- **Tag note.** Carries `IP-CD` (Communication and Documentation) per Topic 2 README — overhead paging and code activation are documentation-and-communication acts; `state-variance` because the operative code-to-meaning mapping is determined by state hospital association and individual health-system policy rather than federal standard.

### Tricky distractors

- **Code Blue — call before compressing?** Some students answer "begin CPR" before "call the code." Both are immediate; the AHRQ case framing emphasizes activating the code team and beginning CPR concurrently — the code call summons the resuscitation team while the bedside nurse begins compressions [s4]. NCLEX items typically credit "call the code" and "begin CPR" both as correct first actions; the wrong-answer pattern is "leave the patient to find a phone."
- **Code Silver vs Code Gray.** The most common code-color confusion. Silver = weapon involved or active shooter; Gray = combative person without a weapon. Wrong-answer pattern: calling Code Gray when a firearm has been identified.
- **Code Black — bomb threat vs death.** A small number of hospitals use Code Black for patient death or for mass casualty; the dominant HASC convention is bomb threat [s2]. NCLEX items use Code Black = bomb threat unless the question stem specifies otherwise.
- **Code Pink — infant vs child.** HASC reserves Pink for infants under 1 year and Purple for children 1+ years [s2]. Some hospitals merge them. Wrong-answer pattern: calling Code Pink for a 6-year-old elopement (would be Code Purple in HASC, or Code Adam at facilities using the Walmart-derived missing-child convention).
- **Code Red — RACE before PASS.** RACE (Rescue, Alarm, Confine, Extinguish) is the response sequence; PASS (Pull, Aim, Squeeze, Sweep) is the extinguisher technique used during the Extinguish step of RACE. Wrong-answer pattern: starting with PASS — fire response begins with rescuing patients in immediate danger, not with grabbing the extinguisher.
- **Code Orange — evacuate vs contain.** Nurses without hazmat training and PPE should evacuate the immediate area, not attempt containment. Wrong-answer pattern: "Place absorbent material on the spill" without first establishing PPE adequacy and evacuation.
- **"Run/Hide/Fight" order.** Run is preferred when a safe egress route exists; Hide is the second choice when egress is blocked; Fight is last resort [s7]. Wrong-answer pattern: hiding when an exit is clearly available, or fighting before exhausting Run and Hide options.
